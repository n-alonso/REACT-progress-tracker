# Show Tracker

## Try It Out

- Clone the project
```
git clone <repository_url> progress-tracker
```
- Install dependencies
```
cd progress-tracker && npm install
```
- Start front and back services
```
npm run dev
```
- Open the project in the browser
```
http://localhost:5173/
```

## Tech Stack

**Language:**
- TypeScript

**Frontend:**
- Vite
- React
- React Router
- React Query
- Mantine
- Tabler Icons

**Backend:**
- Node.js
- Express
- Prisma
- SQLite
